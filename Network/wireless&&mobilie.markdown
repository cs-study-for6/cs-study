## Wireless
wireless newtwork 구성요소
- base station
    - 주로 유선 네트워크에 연결
    - Relay:Host가 data를 전송하면 base station이 받아서 다음 네트워크로 전달해서 목적지 host까지 전달해주는 기능
- wireless hosts
    - 노트북, 스마트폰, IoT 등의 것
    - mobile 일 수도 있고, stationary 일 수도 있다.
